📊 Dashboard – Technology Usage & Trends (IBM Cognos Analytics)
Ce dépôt contient les captures d’écran et la documentation du tableau de bord réalisé dans IBM Cognos Analytics à partir du fichier de données survey_data_updated.csv.
Le tableau de bord est structuré en trois onglets, chacun utilisant un modèle 2×2 et des visualisations adaptées aux métriques demandées.

🧩 1. Current Technology Usage
Cet onglet présente les technologies actuellement utilisées par les répondants.
Visualisations incluses :
- Top 10 LanguageHaveWorkedWith — Bar chart
- Top 10 DatabaseHaveWorkedWith — Column chart
- Top 10 PlatformHaveWorkedWith — Word cloud
- Top 10 WebFrameHaveWorkedWith — Hierarchy bubble chart
Caractéristiques :
- Valeurs triées en ordre décroissant
- Limite Top 10 appliquée
- Titres clairs et étiquettes de valeurs activées
- Couleurs cohérentes pour faciliter la lecture

🚀 2. Future Technology Trends
Cet onglet présente les technologies que les répondants souhaitent utiliser à 
l’avenir.
Visualisations incluses :
- Top 10 LanguageWantToWorkWith — Bar chart
- Top 10 DatabaseWantToWorkWith — Column chart
- Top 10 PlatformWantToWorkWith — Tree map
- Top 10 WebFrameWantToWorkWith — Hierarchy bubble chart
Caractéristiques :
- Utilisation des champs Bars / Length / Color selon le type de graphique
- Tree map configurée avec Area hierarchy, Size et Heat
- Hiérarchie visuelle claire pour les frameworks web
👥 3. Demographics
Cet onglet présente les caractéristiques démographiques des répondants.
Visualisations incluses :
- Respondent distribution by Age — Pie chart
- Respondent Count by Country — Map chart
- Respondent distribution by Formal Education Level — Line chart
- Respondent Count by Age (classified by Education Level) — Stacked bar chart
Caractéristiques :
- Pourcentage affiché dans le pie chart
- Carte configurée avec Regions-Locations et Location color
- Line chart avec markers et value labels
- Stacked bar chart segmenté par niveau d’éducation

📝 Objectif du projet
Ce tableau de bord a été créé dans le cadre du laboratoire portant sur :
- la visualisation de données,
- la compréhension des tendances technologiques,
- l’utilisation de modèles 2×2 dans IBM Cognos Analytics,
- l’application de filtres, tris, limites et titres pour produire des visualisations professionnelles.
